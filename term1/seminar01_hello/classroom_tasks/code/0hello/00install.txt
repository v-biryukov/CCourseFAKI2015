

Как компилировать программы на Windows? Пошаговая инструкция.



0)  Убедитесь, что Windows показывает расширения файлов.

    Расширения файлов это конечная часть имени файла, которая отделяется точкой и обычно говорит о типе файла.
    Например, файлы с текстом обычно имеют расширение .txt, файлы изображений могут иметь расширения .jpg или .png и другие.
    Файлы исходного кода языка C имеют расширения .c
    Так операционная система может понять, что hello.jpg это какое-то изображение, а hello.c это программа на языке C.

    Однако, Windows может не показывать вам эти расширения, а просто отображать в обоих случаях название hello без расширения.
    Это может привести к ошибкам, вы можете назвать ваш файл hello.c, а он может на самом деле называться hello.c.txt
    Чтобы файл можно было скомпилировать он обязательно должен иметь расширение .c и файл hello.c.txt не скомпилируется.

    Что сделать так, чтобы расширения всегда показывались нужно сделать зайти в
        Пуск -> Панель управления -> Параметры Проводника
    появится окошко под названием "Параметры Проводника". В этом окошке нужно выбрать вкладку Вид.
    Во вкладке Вид есть список Дополнительные параметры. В этом списке нужно убрать галочку с пункта 
    "Скрывать расширения для зарегистрированных типов файлов" и нажать ОК.

    Если у вас на Windows стоит английский язык, то так:
        Start -> Control Panel -> Folder Options -> View -> Hide file extensions for known file types



1)  Установить компилятор MinGW, это клон gcc для Windows. Желательно устанавливать 64-х битную версию. Проще всего это сделать так:

        а)  Установить утилиту msys2, она используется для установки других програм
            Загуглите "установить msys2"
            Запомните папку куда вы устанавливаете msys2. 

            !!  Путь до этой папки не должен содержать пробелов, русских символов и другого.
                Только латиница и цифры. 

            По умолчанию установка произойдёт в папку C:\msys2


        б)  Установите компилятор mingw c помощью инструментов msys2
            Загуглите "msys2 установить mingw" или "msys2 установить gcc"


        в)  Найти ту папку, куда установился gcc. Эта папка обязательно должна содержать файл gcc.exe
            По умолчанию папка должна быть C:\msys2\mingw64\bin


        г)  Добавить эту папку в переменную среды Path. Для этого нужно сделать следующее:
            
            Перейти в:
                Пуск -> Панель управления -> Система -> Дополнительные параметры системы     -> Переменные среды
                Start-> Control Panel     -> System  -> Advanced system settings             -> Environment Variables

            Появится окошко под названием "Переменные среды". В любом из списков списке нужно найти переменную Path.
            Далее нужно нажать на переменную Path и выделить её, потом нажать Изменить и появится новое окошко подназванием "Изменить переменную среды". 
            В этом окошке нужно нажать "Создать" и вставить туда путь до папки, содержащий файл gcc.exe и нажать на OK.

            Установить переменную среды нужно, чтобы можно было вызывать gcc из Командной строки.


        д)  Открыть Командную строку. Если у вас уже была открыта Командная строка, то нужно её закрыть и открыть снова.
            Чтобы это сделать можно нажать на Пуск и начать печатать "Командная строка" и найдётся приложение Командная строка.
            Командная строка также называется как cmd.

            Командную строку также часто могут называть как Терминал.


        е)  Чтобы проверить, что вы сделали всё правильно можно написать в Командной строке gcc и нажать Enter.
            Командная строка должна пройтись по всем папкам, записанным в переменной Path и найти в этих папках файл gcc.exe

            Если всё прошло успешно, то должно напечататься

                gcc: fatal error: no input files
                compilation terminated.

            Это означает, что gcc нашёлся, но для компиляции он требует файлы исходного кода.





2)  Как использовать компилятор gcc

    а)  Создайте свою папку, в которой будут хранится ваши файлы исходного кода.
        Для примера пусть путь будет  C:\programming

        !!  Путь до этой папки не должен содержать пробелов, русских символов и другого.
            Только латиница и цифры. 


    б)  Создайте в этой папке файл исходного кода
        Для примера пусть файл будет называться hello.c, то есть полный путь до файла будет C:\programming\hello.c
        Убедитесь, что у файла правильное расширение и он называется именно hello.c, а не hello.c.txt.

        !!  Название файла также не должено содержать пробелов, русских символов и другого.



    в)  Откройте этот файл с помощью текстового редактора.
        Нажмите на файле правой кнопкой, выберите пункт Открыть с помощью и выберети любой текстовый редактор, например Блокнот.
        Если нужного текстового редактора в списке нет, то можно найти его на диске, если выбрать пункты
        Открыть с помощью -> Выбрать другое приложение 

        Поместите в файл hello.c программу:

                #include <stdio.h>

                int main() 
                {
                    printf("Hello World\n");
                }

        Сохраните файл. Откройте файл заново и убедитесь, что текст программы сохранился.


    г)  Зайдите в командную строку


    д)  Перейдите в Командной строке в только что созданную папку.
        В командной строке, также как и в Проводнике, вы всегда находитеся в какой-то папке.

        Чтобы посмотреть в какой папке вы сейчас находитесь, напишите в Командной строке команду  cd  и нажмите Enter.
        Чтобы перейти в другую папку после команды  cd  через пробел укажите название папки, например

            cd C:\programming


    е)  Убедитесь, что вы находитесь в правильной папке
        Для этого можно использовать команду cd без указания папки.
        Также можно использовать команду dir, чтобы посмотреть содержимое папки. Папка должна содержать один файл под названием hello.c


    ё)  Скомпилируйте программу hello.c с помощью компилятора gcc.
        Для этого в Командной строке нужно написать

            gcc hello.c

        Если компиляция пройдёт успешно, то в этой же папке появится файл под названием a.exe
        Запустите этот файл, написав в Командной строке его имя

            a.exe

        Или можно даже без расширения:

            a

        Программа должна напечатать на экран (в Командную строку) фразу Hello World!



    ж)  Можно объединить компиляцию и запуск программы и делать это в одной строке вот так:

            gcc hello.c && a.exe

        Постоянно вводить эту команду не нужно. Находясь в Командной строке можно нажать клавишу Стрелка Вверх, чтобы
        повторить предыдущую команду. Стрелку Вверх можно нажимать много раз.





3)  Установите удобный текстовый редактор для программирования
    Советую редактор Sublime Text (www.sublimetext.com)

    Сделайте так, чтобы программы исходного кода языка C автоматически открывались в программе Sublime Text.
    Для этого нужно сделать следующее:

        а)  нажмите правой кнопкой на любом файле с расширением .c

        б)  выбрать Свойства

        в)  нажать Изменить и выбрать редактор Sublime Text

    Теперь все файлы с расширением .c будут автоматически открываться в редакторе Sublime Text
