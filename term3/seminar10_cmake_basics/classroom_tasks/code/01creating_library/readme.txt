Привет! Это простейший проект с использованием CMake.

Для сборки этого проекта в терминале зайдите в папку build и исполните:

    cmake ..
    cmake --build .

После этого в папке build появится исполняемый файл.

Файл CMakeLists.txt - это основной файл проекта. 
Он содержит код написанный на языке CMake и описывает как нужно собирать тот или иной проект.
Данный проект очень простой, поэтому код содержит всего 3 строки:

    cmake_minimum_required(VERSION 3.20)
    project(helloProject)
    add_executable(hello hello.cpp)

cmake_minimum_required  -   задаём минимальную возможную версию CMake-а.
                            попытка компиляции с помощью CMake-а меньшей версии
                            приведёт к ошибке.

project                 -   устанавливаем имя проекта
add_executable          -   задаём исполняемый файл
